Movie list api

Get movie Information


To get the info from every movie on the list you would use this url


/api/movies


To get the information from one movie use this url:

/api/:id/movie

Use an actual id number instead of :id



Get actor information


To get information on the entire list of actors You use this url


/api/actors


To get the information for one of the actors you would use this


/api/:actor_id/actor

Use an actual id number instead of :actor_id
